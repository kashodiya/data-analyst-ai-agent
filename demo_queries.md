# Demo Queries for Screenshots

## After connecting to Chinook database (`../data/chinook.db`):

### Query 1: Top Customers
```
Show me the top 5 customers by total purchases
```

### Query 2: Sales Analysis
```
What are the total sales by country?
```

### Query 3: Music Trends
```
Which music genres generate the most revenue?
```

### Query 4: Employee Performance
```
Show me the sales performance of each employee
```

### Query 5: Time-based Analysis
```
Display monthly sales trends for the last year
```

## Tips for Screenshots:

1. **Main Interface**: Take this right after opening http://localhost:8080
2. **Database Connection**: Click the database icon and enter `../data/chinook.db`
3. **Query Response**: Use Query 1 above for the best visual result
4. **Session History**: Create 2-3 sessions with different queries first
5. **Tool Usage**: Click "Show Tool Usage Details" on any response
